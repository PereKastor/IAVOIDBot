v0.24  22/09/2016   (Current Release)
--------------------------------------- 

- PlayStandaloneMatch can define LoadTime, InitTime and TurnTime to help debugging
- Generating solution process now extracts folder/subfolders tree structure and replicates it into VS Studio Filters automatically
- Create job now use a filelist.txt file containing the list of all mandatory files to zip. This file is automatically generated/updated by the solution generation process.
- TeamInfo.team now requires to set the AIBootCampEngine path and allows to define Python path if 'where python' detection process isn't working (ex. having more than one python installation)
- Fix CheckInstall issue with unsupported characters in file path (parenthesis, spaces...) 
- PlayStandaloneMatch results LogID is now generated using date-time info
- Fix CleanAll and rebuild_vis_sln_x64.bat annoying timing issue when deleting build folder. It is now instant.
- Improve Documentation
